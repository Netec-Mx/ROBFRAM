# Robot Framework y Automatización de Pruebas

## Manual Teórico del Curso

Preparación para la certificación **RFCP** (Robot Framework Certified Professional)

---

## Información del curso

| Campo | Detalle |
|---|---|
| **Versión de la tecnología** | Robot Framework 7.x · Python 3.12 |
| **Duración total** | 35 horas (9 sesiones) |
| **Distribución** | 40% teoría (14 h) · 60% práctica (21 h) |

## Descripción general

Curso práctico de automatización de pruebas y RPA con Robot Framework. Cubre desde fundamentos hasta técnicas avanzadas: automatización web, API, RPA, BDD, extensión con Python y preparación para la certificación RFCP.

## Audiencia

Ingenieros de QA, analistas de pruebas, desarrolladores y profesionales de automatización con conocimientos básicos de programación. Equipos de TI en el sector telecomunicaciones que buscan adoptar automatización de pruebas y RPA con estándares de calidad.

## Objetivos del curso

Al finalizar el curso, el participante será capaz de:

- Diseñar, implementar y mantener suites de prueba automatizadas con Robot Framework.
- Aplicar patrones BDD orientados al negocio.
- Crear librerías personalizadas en Python.
- Automatizar procesos RPA de extremo a extremo.
- Ejecutar pipelines de prueba en entornos CI/CD.

## Prerrequisitos

Conocimientos básicos de programación (Python recomendado). Manejo básico de línea de comandos. Familiaridad con conceptos de testing de software. Requerimientos tecnológicos: Python 3.10+, VS Code, acceso a internet.

## Cómo usar este manual

Este manual cubre la **teoría** de cada capítulo — los conceptos, la sintaxis y el razonamiento detrás de cada técnica. La **práctica guiada** (paso a paso, con código ejecutable) vive en la Guía del Estudiante (`robfram-guide`), y las diapositivas de cada sesión en `robfram-ppts`. Los tres materiales están alineados capítulo por capítulo.
